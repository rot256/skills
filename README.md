# Skills 

Skills I use for writing/reading code.
These skills write code in ways that are very opionated,
for instance, the python programming skill enforced static type-checking,
has strict choices for dependencies, and uses pydantic consistently for input/schema-validation.
If you like writing your python code more "duck-typed", then this is probably not the skill for you.

In general, the skills (and my AGENTS.md), lean towards pendantic and formal:
I want most/all my code to look the same, have the same dependencies, etc.
Use at your own peril.
